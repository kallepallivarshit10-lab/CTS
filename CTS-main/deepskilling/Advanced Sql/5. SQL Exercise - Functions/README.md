
# Employee Management System - SQL Functions

## Objective

The objective of this experiment is to understand and implement different types of SQL Server User-Defined Functions (UDFs). The exercises demonstrate Scalar Functions, Table-Valued Functions, Nested Functions, function modification, deletion, and execution using Microsoft SQL Server.

---

## Software Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

---

## Database Tables

- Departments
- Employees

---

## Exercises Performed

### Exercise 1
Created a Scalar Function to calculate the annual salary of an employee.

### Exercise 2
Created a Table-Valued Function to return employees belonging to a specific department.

### Exercise 3
Created a User-Defined Function to calculate employee bonus (10% of salary).

### Exercise 4
Modified the bonus calculation to 15% of salary.

### Exercise 5
Deleted the User-Defined Function.

### Exercise 6
Executed the Annual Salary Scalar Function.

### Exercise 7
Returned the annual salary for a specific employee.

### Exercise 8
Executed the Table-Valued Function to retrieve employees from the Finance department.

### Exercise 9
Created a Nested Function to calculate total compensation using Annual Salary and Bonus.

### Exercise 10
Modified the Nested Function to use a new bonus calculation.

## Advantages of SQL Functions

- Improves code reusability.
- Simplifies complex calculations.
- Makes SQL queries easier to read.
- Reduces code duplication.
- Can be reused across multiple queries.
