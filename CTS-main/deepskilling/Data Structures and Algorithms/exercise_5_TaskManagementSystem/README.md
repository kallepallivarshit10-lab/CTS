# Exercise 5: Task Management System

## Aim

To implement a Task Management System using a Singly Linked List in C# and perform add, search, traverse, and delete operations.

---

## Scenario

A task management system needs to efficiently manage tasks using a linked list.

---

## Classes Created

### Task.cs

Represents a task with:
- Task ID
- Task Name
- Status
- Next node reference

### TaskLinkedList.cs

Provides methods to:
- Add Task
- Search Task
- Traverse Tasks
- Delete Task

### Program.cs

Tests all linked list operations.

---

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(n) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

---

## Advantages of Linked Lists

- Dynamic size
- Efficient insertion and deletion
- No contiguous memory allocation required

---

## Limitations

- No direct index-based access
- Extra memory required for storing node references

---

## Result

Successfully implemented a Task Management System using a Singly Linked List in C# and analyzed its performance.
