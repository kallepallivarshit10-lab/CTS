# Exercise 3: Sorting Customer Orders

## Aim

To implement Bubble Sort and Quick Sort algorithms in C# to sort customer orders based on their total price.

---

## Scenario

An e-commerce platform needs to sort customer orders by total price so that high-value or low-value orders can be processed efficiently.

---

## Classes Created

### Order.cs

Represents an order with:
- Order ID
- Customer Name
- Total Price

### SortingOperations.cs

Implements:
- Bubble Sort
- Quick Sort

### Program.cs

Creates sample orders, sorts them using both algorithms, and displays the results.

---

## Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|----------|-------|
| Bubble Sort | O(n) | O(n²) | O(n²) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) |

---

## Result

Successfully implemented Bubble Sort and Quick Sort algorithms in C# and compared their performance.
