
# Exercise 6: Library Management System

## Aim

To implement Linear Search and Binary Search algorithms in C# for searching books by title.

---

## Scenario

A library management system allows users to search books by title efficiently.

---

## Classes Created

### Book.cs

Represents a book with:
- Book ID
- Title
- Author

### SearchOperations.cs

Implements:
- Linear Search
- Binary Search

### Program.cs

Tests both search algorithms.

---

## Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|----------|-------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

---

## Comparison

- Linear Search works on unsorted data.
- Binary Search requires sorted data.
- Binary Search is much faster for large datasets.

---

## Result

Successfully implemented Linear Search and Binary Search in C# and compared their performance.
