# Exercise 4: Employee Management System

## Aim

To implement an Employee Management System using arrays in C# and perform add, search, traverse, and delete operations.

---

## Scenario

A company needs to efficiently manage employee records using an array.

---

## Classes Created

### Employee.cs

Represents an employee with:
- Employee ID
- Name
- Position
- Salary

### EmployeeManagement.cs

Provides methods to:
- Add Employee
- Search Employee
- Traverse Employees
- Delete Employee

### Program.cs

Tests all employee management operations.

---

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

---

## Advantages of Arrays

- Fast index access
- Simple implementation
- Memory efficient for fixed-size collections

---

## Limitations of Arrays

- Fixed size
- Slow insertion and deletion
- Requires shifting elements after deletion

---

## Result

Successfully implemented an Employee Management System using arrays in C# and analyzed the performance of array operations.
