# Exercise 2: E-commerce Platform Search Function

## Aim

To implement Linear Search and Binary Search algorithms in C# and compare their performance.

---

## Scenario

An e-commerce platform needs efficient searching of products based on Product ID.

---

## Classes Created

### Product.cs

Stores:

- Product ID
- Product Name
- Category

### SearchOperations.cs

Implements:

- Linear Search
- Binary Search

### Program.cs

Tests both searching algorithms.

---

## Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|----------|-------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

---

## Result

Successfully implemented Linear Search and Binary Search using C# and compared their performance.
