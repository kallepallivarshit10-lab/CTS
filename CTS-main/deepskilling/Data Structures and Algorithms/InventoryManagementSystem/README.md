# Exercise 1: Inventory Management System

## Aim

To implement an Inventory Management System using C# and Dictionary for efficient storage and retrieval of products.

---

## Scenario

A warehouse stores product information. The system supports adding, updating, deleting, and displaying products.

---

## Data Structure Used

Dictionary<int, Product>

Reason:
- Fast insertion
- Fast searching
- Fast updating
- Fast deletion

---

## Classes Created

### Product.cs

Represents a product with:
- Product ID
- Product Name
- Quantity
- Price

### Inventory.cs

Provides methods to:
- Add Product
- Update Product
- Delete Product
- Display Products

### Program.cs

Demonstrates inventory operations.

---

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) |
| Search | O(1) |
| Update | O(1) |
| Delete | O(1) |

---

## Expected Output

- Products are added successfully.
- Product details are displayed.
- Product quantity is updated.
- Product is deleted successfully.

---

## Result

Successfully implemented an Inventory Management System using C# and Dictionary.
