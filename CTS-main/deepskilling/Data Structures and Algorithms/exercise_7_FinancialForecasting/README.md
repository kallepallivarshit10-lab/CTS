# Exercise 7: Financial Forecasting

## Aim

To implement a recursive algorithm in C# to predict future financial values based on a fixed annual growth rate.

---

## Scenario

A financial forecasting tool predicts future values using the current value, annual growth rate, and the number of years.

---

## Classes Created

### Forecast.cs

Implements a recursive method to calculate future value.

### Program.cs

Tests the recursive forecasting algorithm.

---

## Time Complexity

- Time Complexity: O(n)
- Space Complexity: O(n)

---

## Advantages of Recursion

- Simple and readable code
- Suitable for problems that can be divided into smaller subproblems
- Easy to understand for mathematical definitions

---

## Optimization

- Memoization
- Dynamic Programming
- Iterative approach for better memory usage

---

## Result

Successfully implemented a recursive financial forecasting algorithm in C# and analyzed its performance.
